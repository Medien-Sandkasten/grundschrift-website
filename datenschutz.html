3. Allgemeines zur Datenverarbeitung

Die Grundschrift App ist eine Lern-App für Kinder im Grundschulalter zum Üben der Handschrift. Sie kann in zwei Modi betrieben werden:

  • Lokaler Modus — Alle Daten werden ausschließlich auf dem Endgerät gespeichert. Es findet keine Datenübertragung an Server statt.
  • Schulmodus — Die App wird mit einem Schulserver synchronisiert, damit Lehrkräfte den Lernfortschritt einsehen können.

Im lokalen Modus werden keine personenbezogenen Daten an uns übermittelt. Die nachfolgenden Abschnitte beziehen sich daher, soweit nicht anders angegeben, auf die Nutzung im Schulmodus.

5. Verschlüsselung und Privacy by Design

Die Grundschrift App setzt konsequent auf Privacy by Design gemäß Art. 25 DSGVO:

  • Client-seitige Verschlüsselung: Namen von Kindern, Gruppen und Lehrkräften werden vor der Übertragung auf dem Endgerät mit AES-CBC verschlüsselt. Als Schlüssel dient der Schulschlüssel, der ausschließlich der jeweiligen Schule bekannt ist.
  • Kein Zugriff durch den Betreiber: Der Schulschlüssel wird zu keinem Zeitpunkt an den Server übermittelt oder dort gespeichert. Der Betreiber des Servers kann die verschlüsselten personenbezogenen Daten daher nicht entschlüsseln und nicht lesen.
  • Passwort-Hashing: Kinder-Passwörter werden als SHA-256-Hash gespeichert. Das Klartext-Passwort verlässt zu keinem Zeitpunkt das Endgerät.

6. Hosting und Datenübermittlung

Der Server wird bei der Hetzner Online GmbH in Deutschland betrieben. Eine Übermittlung personenbezogener Daten in Drittländer außerhalb der EU / des EWR findet nicht statt.

Die Kommunikation zwischen App und Server erfolgt ausschließlich über eine TLS-verschlüsselte Verbindung (HTTPS).

7. Drittanbieter-Dienste

Die App verwendet keine Analyse-, Tracking- oder Werbedienste. Insbesondere werden keine Dienste wie Google Analytics, Firebase Analytics, Crashlytics, Sentry oder vergleichbare Tools eingesetzt. Es werden keine Cookies gesetzt.

8. Speicherdauer

  • Lokaler Modus: Daten verbleiben auf dem Gerät, bis die App deinstalliert oder die Daten manuell gelöscht werden.
  • Schulmodus: Daten werden auf dem Server gespeichert, solange die Schule die App aktiv nutzt. Nach Beendigung der Nutzung werden die Daten auf Anfrage der Schule vollständig gelöscht.

Personenbezogene Daten der Kinder (insbesondere Name und Gruppenzugehörigkeit) werden client-seitig verschlüsselt, sodass sie auf dem Server nicht im Klartext vorliegen. Es werden keine Daten erhoben, die über den pädagogischen Zweck hinausgehen.
